# BackPack Hero

## Description
Développement d’une version simplifiée du jeu *Backpack Hero* dans le cadre du module de Programmation Orientée Objet (L3 Informatique).

Le jeu consiste à explorer un donjon, gérer un inventaire limité (sac à dos) et affronter des ennemis dans un système de combat au tour par tour.

## Fonctionnalités
- Exploration d’un donjon composé de plusieurs salles
- Système de combat au tour par tour (attaque, défense, gestion d’énergie)
- Gestion d’un inventaire avec contraintes d’espace (sac à dos)
- Intégration d’équipements (armes, armures, objets)
- Gestion des interactions entre objets et effets de combat
- Interface graphique simple basée sur le modèle MVC

## Technologies
- Java
- Programmation Orientée Objet (POO)
- Modèle MVC (Modèle-Vue-Contrôleur)
- Bibliothèque graphique Zen

## Contexte du projet
Projet réalisé en binôme dans le cadre de la Licence 3 Informatique.

Développement en plusieurs phases :
- Phase 1 : implémentation des bases du jeu
- Phase 2 : ajout des interactions, effets et progression
- Phase 3 : version complète avec génération et fonctionnalités avancées

## Résultat
Projet académique validé avec une note de 14/20.

## Objectifs pédagogiques
- Concevoir une architecture orientée objet propre et modulaire
- Implémenter des mécaniques de jeu complexes
- Structurer un projet en respectant des contraintes académiques
- Développer une interface graphique simple
